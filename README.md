# ☁️ Google Cloud Gen AI DevOps Assistant

An intelligent, serverless DevOps assistant designed to help developers instantly interpret and resolve cloud deployment failures, invalid shell scripts, and broken container pipelines.

## 🚀 Features
- **Instant Log Parsing:** High-speed scanning of verbose terminal output to capture the precise point of system failure.
- **Automated Script Rewriting:** Automatic correction of invalid operators, missing parameters, or broken paths in deployment configurations.
- **Interactive Context Chat:** Allows developers to ask follow-up questions regarding structural container dependencies or runtime environments.

## 🛠️ Technology Stack
- **Frontend/UI:** Streamlit (Python)
- **Deployment Platform:** Google Cloud Run (Fully Containerized, Serverless)
- **AI Core Engine:** Google Cloud Vertex AI & Gemini API
